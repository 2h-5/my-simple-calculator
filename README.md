## Stories Behind the Work

I chose to build this calculator app (*game*) is because of **curiosity** on developing more features <u>based on my assignments</u> from one of my university courses —— HCI Design.
What I have further developed are:
1. **"Enhance"** the calculator to be more intuitive on graphical design, button interaction, and displaying operating method.
2. Try to build the calculator using **Swift** (programming language developed by *Apple*) ***and/or*** deploy a **<u>frontend framework for Swift</u>** using **Windows**.

In the final version, I enhanced the calculator and deploy it using frontend for Swift.
